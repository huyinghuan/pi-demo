# GO raspberry pi Sensor Demo

Go 树莓派的控制硬件demo， 附电路图

## compile 编译

gox or go build

gox https://github.com/mitchellh/gox

gox build:
```
 gox -osarch="linux/arm" -output="bin/xxx"
```

go build：

```
 export GOOS=linux
 export GOARCH=arm
 go build ...
```
